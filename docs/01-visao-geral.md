# Visão Geral

Sistema interno web para suportar a operação de compra, receção, armazenagem, montagem e controlo de cabazes.

## Objetivos

- controlar stock por movimentos
- garantir rastreabilidade por lote e validade
- permitir montagem parcial
- permitir substituição controlada de componentes
- manter stock de cabaz montado
- suportar várias localizações
- alertar para mínimos e expiração
